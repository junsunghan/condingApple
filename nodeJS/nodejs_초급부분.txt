nodeJs 시작하는 방법

1. nodeJS 설치
2. 터미널에 npm init 을 이용해 package.json 만들기
3. 터미널에 npm init 을 입력하고 enter를 치다가 entry point가 나왔을 때 server.js 입력 -> main 파일을 지정한것
4. 그 후 enter 계속 치기
5. npm install express로 express 라이브러리 설치
6. npm install -g nodemon 라이브러리 설치 - 라이브러리 서버 껏다 키는 작업 자동화
7. nodemon server.js 로 실행
8. 만약 오류가 생기면 powershell 관리자 권한으로 실행
9. executionpolicy 엔터 -> set-executionpolicy unrestricted 엔터 -> y 엔터
